**Access Rights Summary:**

| Model | Rental User | Rental Manager |
|-------|-------------|----------------|
| **Equipment** | Read only | Full access |
| **Categories** | Read only | Full access |
| **Serials** | Read only | Full access |
| **Projects** | Read/Write/Create | Full access |
| **Project Items** | Read/Write/Create | Full access |
| **Status History** | Read/Create | Full access |
| **Return Wizard** | Full access | Full access |

**Notes:**
- Users can create and edit their own projects but cannot delete
- Users can create status history (for tracking) but not edit/delete
- Both groups have full wizard access (needed for returns)
- Managers have unrestricted access to everything
